📰 Blog de Posts - Tech Challenge
📌 Sobre o Projeto

Este projeto é um Blog de Posts desenvolvido em React como parte do Tech Challenge.

A aplicação permite visualizar posts, buscar conteúdos e gerenciar publicações através de um painel administrativo.

🚀 Tecnologias Utilizadas

React

React Router DOM

JSON Server

JavaScript

CSS

⚙️ Funcionalidades

✔ Listagem de posts
✔ Visualização de post completo
✔ Busca de posts
✔ Criação de novos posts
✔ Edição de posts
✔ Exclusão de posts
✔ Painel administrativo
✔ Navegação entre páginas

▶️ Como Executar o Projeto
1️⃣ Instalar dependências
npm install
2️⃣ Iniciar API fake
npx json-server --watch db.json --port 3001
3️⃣ Iniciar aplicação
npm start

Aplicação disponível em:

http://localhost:3000
